Este repositorio contiene todos los proyectos, retos de programación, laboratorios y ejercicios prácticos desarrollados durante el Bootcamp de Python en el marco del programa **MisiónTIC 2022**.

## 📌 Objetivo del Programa
Formar programadores con competencias técnicas en el lenguaje Python, abarcando desde los fundamentos de la lógica de programación hasta el desarrollo de aplicaciones funcionales y la manipulación de datos.

## 🗂️ Estructura del Repositorio

El contenido está organizado por ciclos y semanas de aprendizaje:

*   `/Ciclo_1_Fundamentos`: Introducción a la algoritmia y lógica básica.
*   `/Ciclo_2_Programacion_Basica`: Estructuras de datos y Programación Orientada a Objetos (POO).
*   `/Retos`: Soluciones individuales a los retos oficiales calificables del programa.
*   `/Proyectos`: Proyectos integradores finales del bootcamp.

## 🚀 Contenido Académico

### Semana 1: Introducción y Sintaxis Básica
*   Variables, tipos de datos primitivos (int, float, string, bool).
*   Operadores aritméticos, lógicos y de comparación.
*   Lectura y salida de datos por consola (`input()` y `print()`).

### Semana 2: Estructuras de Control
*   Condicionales (`if`, `elif`, `else`).
*   Ciclos iterativos y de control (`while`, `for`).
*   Manejo de excepciones básico (`try`, `except`).

### Semana 3: Colecciones y Estructuras de Datos
*   Listas y tuplas (manipulación y métodos).
*   Diccionarios y conjuntos (sets).
*   Comprensión de listas (List Comprehension).

### Semana 4: Funciones y Modularidad
*   Definición de funciones y paso de parámetros (`args` y `kwargs`).
*   Funciones Lambda, `map()`, `filter()` y `reduce()`.
*   Creación e importación de módulos locales.

### Semana 5: Programación Orientada a Objetos (POO)
*   Clases, objetos, atributos y métodos.
*   Pilares de POO: Encapsulamiento, Herencia y Polimorfismo.
*   Métodos especiales (constructores).

## 🛠️ Tecnologías y Herramientas Utilizadas

## 🔧 Instalación y Ejecución

Para correr los scripts de este repositorio localmente, sigue estos pasos:

1. Clona este repositorio en tu máquina local:
   ```bash
   git clone https://github.com
   ```

2. Navega al directorio del proyecto:
   ```bash
   cd bootcamp-python-misiontic
   ```

3. Ejecuta el archivo que desees (ejemplo con un script de la semana 1):
   ```bash
   python Ciclo_1_Fundamentos/Semana_1/ejercicio_practico.py
   ```
